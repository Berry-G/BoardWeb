# BoardWeb
Springboot project 연습

## START
https://start.spring.io/ 에서 프로젝트 생성
## Dependencies(pom.xml)
- Spring Boot Dev Tools
- Lombok
- Spring Web
- MyBatis Framework
- PostgreSQL Driver
- jsp 사용을 위해 japer 추가
```
<dependency>
  <groupId>org.apache.tomcat.embed</groupId>
  <artifactId>tomcat-embed-jasper</artifactId>
  <version>11.0.0-M7</version>
</dependency>
```
